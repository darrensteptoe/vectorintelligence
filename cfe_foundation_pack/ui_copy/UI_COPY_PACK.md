## Core helper text
### Total Planned Budget
“The full projected campaign budget under the active scenario.”

### Required Budget
“The portion of the plan currently marked as necessary to execute the selected strategy.”

### Optional Budget
“Costs the campaign may still want, but can delay or remove if the path tightens.”

### Peak Spend Window
“The month or phase where projected cost pressure is highest.”

### Current Raise Pace
“How the campaign’s actual intake is tracking relative to the active period target.”

### Next Checkpoint Target
“What must be raised by the next practical checkpoint to keep the path workable.”

### Reserve Status
“How much cushion the campaign currently has before the next major spending window.”

### Field Funding Status
“Whether the selected field posture is safely supported under the current finance path.”

## Empty states
### No budget lines yet
“No budget lines have been added yet. Start by entering the campaign’s known costs rather than aiming for a perfect final budget on the first pass.”

### No benchmark set yet
“No benchmark set has been generated yet. Historical evidence is optional for planning, but it is useful for realism, threshold framing, and report context.”

### No funding path yet
“No funding path has been computed yet. Build or select a budget plan first, then compute the path so the campaign has a time-aware raise target.”

### No activity logged yet
“No finance activity has been logged for this period yet. A clean plan matters, but the engine gets more useful once actual work and outcomes are recorded.”

### No donor intelligence yet
“No donor intelligence summary is available yet. Import or sync contribution data, then review geography and occupation outputs.”

### No active risks
“No active risks are currently above threshold under the active scenario.”

## Confirmation copy
### Import field snapshot
“Import the selected field-demand snapshot into the active campaign scenario? This will not change FPE logic. It will only update the finance-side demand inputs.”

### Recompute funding path
“Recompute the funding path with the latest budget, benchmark, and current-condition inputs?”

### Apply manual classification override
“Apply this manual override? Manual values take precedence over automatic classification until changed.”
