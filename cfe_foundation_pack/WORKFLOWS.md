# Campaign Finance Engine (CFE) — Workflows

## Workflow 1 — Set up campaign and race
1. Create race profile.
2. Create campaign profile.
3. Load election and filing calendars.
4. Select active scenario.

## Workflow 2 — Build full campaign budget
1. Create budget plan.
2. Add all budget lines.
3. Mark required vs optional.
4. Assign phase and timing.
5. Save draft and review totals.

## Workflow 3 — Import field demand
1. Select FPE snapshot.
2. Validate bridge schema.
3. Import field cost signals.
4. Merge into budget timeline.
5. Review field affordability notes.

## Workflow 4 — Ingest historical evidence
1. Select comparable scope.
2. Pull public data.
3. Preserve raw records.
4. Normalize records.
5. Build comparable pool.

## Workflow 5 — Generate benchmark set
1. Review comparables.
2. Confirm included pool.
3. Generate threshold set.
4. Review realism notes.
5. Save benchmark set.

## Workflow 6 — Generate funding path
1. Select budget plan.
2. Select benchmark set.
3. Load current condition.
4. Compute funding requirement snapshot.
5. Review total target, checkpoints, reserve, and path status.

## Workflow 7 — Allocate channel targets
1. Select period target.
2. Set planned channel mix.
3. Generate channel targets.
4. Review concentration warnings.
5. Save channel plan.

## Workflow 8 — Plan finance activity
1. Create finance activities for the period.
2. Assign owners.
3. Set goals.
4. Link activities to channels.
5. Save on finance calendar.

## Workflow 9 — Record outcomes
1. Mark activity outcome.
2. Log asks, pledges, receipts, and follow-up.
3. Record event net yield where relevant.
4. Update task queue.
5. Refresh planned vs actual summaries.

## Workflow 10 — Review performance and adjust
1. Check planned vs actual.
2. Review active risks.
3. Review reserve and field funding status.
4. Adjust next-period targets and activities.
5. Generate report outputs.
